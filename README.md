GLSandbox
=========

A dumb test project to play around with OpenGL and experiment with the architecture of a real-time rendering app.  
Aiming to evolve into a more nicely-featured rendering engine.  

### Current features

* Transforms
* FPS camera & key bindings
* Self-drawing meshes

### Roadmap

* Lighting
* `.obj` file handling
* Shadows
* Scene graph
* PBR
